    Style manipulation

    Single Style Change

    Click the list elements to change the text colors coded to the data-color attribute

    • This will be red
    • This will be green
    • This will be blue 
    $('#list3 li').on('click', function(){
    $(this).css('color', $(this).data('color'))
});

    Multiple changes at once

    Click the list elements to change the text colors & font-sizes

    • This will be red and big
    • This will be green and big
    • This will be blue and big 
    $('#list4 li').on('click', function(){
    $(this).css({
        color: $(this).data('color'),
        fontSize: '22px' // or "font-size": "22px"
    });
});
